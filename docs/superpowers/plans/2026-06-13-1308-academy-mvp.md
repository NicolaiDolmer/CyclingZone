# Akademi-MVP (#1308) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Gøre "udvikling fra egen avl" til en reel mekanik fra dag 1 i frisk sæson 1: hvert menneske-hold får ved relaunch et intake-kuld af 3-5 unge kandidater (16-21), kan signe 0-2 ind i et 8-plads-akademi adskilt fra senior-30-cap'en, akademiryttere træner dagligt med en ungdoms-multiplikator, og afviste kandidater ruller på løbende ungdomsauktioner med free-agent-fallback.

**Architecture:** Akademiryttere bor i `riders` med et `is_academy`-flag og `team_id` = ejende hold; et nyt `academy_intake`-join holder de tilbudte kandidater pr. hold før de signes. Akademiryttere ekskluderes fra ALLE senior-squad-cap-tællinger (det er den kritiske kontrakt-flade) men inkluderes i det daglige trænings-loop med en alders-baseret ungdoms-multiplikator. Et nyt intake-trin i relaunch-orchestratoren (efter starter-squad, før kontrakt-seed) genererer kandidat-kuld pr. menneske-hold, biased på `season_1_identity_basis`. Hele systemet er bag `academy_enabled`-flaget (OFF til relaunch), så det matcher søskende-leverancerne #1305/#1306/#1307.

**Tech Stack:** Node.js/Express (backend), Supabase/Postgres (DB; Supabase MCP `apply_migration` mod disposabel branch — ALDRIG prod), React+Vite (frontend), `node --test` (backend+frontend unit-tests), i18n via namespace-JSON (en+da), determinisme via seeded `makeRng`.

1. **Datamodel = `is_academy`-flag på `riders`** (ikke separat squad-tabel). Akademiryttere har `team_id` = ejende hold + `is_academy=true`. **Ekskluderes fra senior-30-cap'en overalt** (sweep, Task 7). Inkluderes i daglig træning. Genbruger alt eksisterende rider-flow (scouting, kontrakt-felter, finance).
2. **Kun menneske-hold** får akademi/intake (= `getBetaManagerTeams`-filteret; matcher daglig-trænings-filteret + spec afsnit 9.1: AI-hold = kun passiv L0-vækst). AI/bank/frosne hold får intet akademi.
3. **Fuld u-auktion/free-agent-loop** (Fase B): afviste kandidater listes straks som individuelle ungdoms-auktioner (`auctions.is_youth=true`); usolgte → free agent; ny "sign free agent til minimumsløn"-route. System-bølge-*integration* forbliver fast-follow (markeds-pakke #1310).

### Afledte designvalg (følger af spec + 1-3, ikke nye beslutninger)

- **Kandidater = friskt 16-21-kohorte** genereret i intake-trinnet (genbruger `fictionalRiderGenerator`-maskineriet: `makeRng`, `gaussian`, `STAT_KEYS`, navne-/nation-pools — men alders-clamp 16-21, lavere stat-means (raw), bred potentiale-spredning). De udskæres IKKE fra de 800 seniorer (starter-squad-allokeringen røres ikke). `pcm_id=null`.
- **Kandidat-livscyklus:** intake genererer kandidat-ryttere med `team_id=NULL` + `is_academy=false`, linket til holdet via `academy_intake(status='offered')`. **Sign** → `is_academy=true`, `team_id`=hold, kontrakt + signing-fee, intake→`signed`. **Reject** → intake→`rejected` + kandidaten disponeres (Fase A: parkeres som ung free agent `team_id=NULL`; Fase B: ungdomsauktion). Vinder af ungdomsauktion / free-agent-signing → `is_academy=true` i køberens akademi (8-plads-cap gælder).
- **8 pladser:** `count(riders WHERE team_id=X AND is_academy=true) < ACADEMY.SLOTS (8)` før enhver akademi-tilføjelse (sign, auktions-vind, free-agent-sign). Hård cap (ingen soft-buffer).
- **Alder 16-21 + tvunget valg ved 22:** birthdate findes allerede → ingen ekstra kolonne. Selve 22-flowet (promover/sælg/slip) er post-launch; datamodellen er klar.
- **Ungdoms-multiplikator** ganges i `dailyAbilityDelta` (én ekstra faktor). Akademiryttere får `skipGrowth` i sæson-transitionen ligesom menneske-holds seniorer (anti-double-dip).
- **Økonomi-dræn:** akademi-drift pr. plads pr. sæson (opkræves ved sæson-finalization i `economyEngine`) + signing-fee ( engang). Beløb fastlægges i sim-harness (Task 9) og **godkendes af ejer** før flag-flip — ikke autonomt tunet.
- **Flag:** `academy_enabled` i `app_config`, default false. Alt akademi-flow no-op'er når OFF.

### Noterede afhængigheder

- `#1103`-backfill-checklisten (GitHub-issue) skal have intake-trinnet + flag-flip tilføjet (Task 11/16).
- `#1305` daily-training-flag skal være ON for at akademi-træning kører (akademi-multiplikatoren er en no-op hvis daily training er OFF — begge flippes ved relaunch).

```js
// Akademi-flag + konstanter (#1308). Flag-mønster spejler dailyTrainingFlag.js
// (app_config, runtime-flip, fail-safe false). Konkrete beløb (DRIFT_PER_SEASON,
// SIGNING_FEE_RATE) kalibreres i sim-harness (academyEconomySimulation.js) og
// godkendes af ejer før flag-flip — placeholders her er sim-startpunkter.

export const ACADEMY = Object.freeze({
  FLAG_KEY: "academy_enabled",
  SLOTS: 8,                 // pladser pr. akademi (hård cap)
  MIN_AGE: 16,
  MAX_AGE: 21,
  INTAKE_MIN: 3,            // kandidater pr. kuld
  INTAKE_MAX: 5,
  SERIOUS_MIN: 1,           // "seriøse" (høj potentiale) pr. kuld
  SERIOUS_MAX: 3,
  YOUTH_MULT: 1.5,          // ungdoms-multiplikator ved MIN_AGE (aftager mod 1.0 ved 22)
  DRIFT_PER_SEASON: 15_000, // SIM-STARTPUNKT — kalibreres + ejer-godkendes (Task 9)
  SIGNING_FEE_RATE: 0.25,   // signing-fee = rate × market_value. SIM-STARTPUNKT (Task 9)
  SALARY_RATE: 0.10,        // ungdoms-løn = 10% af market_value (spejler kontrakt-seed)
  CONTRACT_LENGTH: 3,       // akademi-kontrakt-længde (sæsoner)
});

export function isAcademyAge(age) {
  return Number.isFinite(age) && age >= ACADEMY.MIN_AGE && age <= ACADEMY.MAX_AGE;
}

// Lineær aftagning fra YOUTH_MULT (ved MIN_AGE) mod 1.0 (ved MAX_AGE+1=22).
// Uden for akademi-alder = 1.0 (ingen boost).
export function youthMultiplier(age) {
  if (!isAcademyAge(age)) return 1.0;
  const span = (ACADEMY.MAX_AGE + 1) - ACADEMY.MIN_AGE; // 22-16 = 6
  const t = (age - ACADEMY.MIN_AGE) / span;             // 0 ved 16 → 5/6 ved 21
  return ACADEMY.YOUTH_MULT - (ACADEMY.YOUTH_MULT - 1.0) * t;
}

export async function isAcademyEnabled(supabase) {
  if (!supabase?.from) return false;
  try {
    const { data, error } = await supabase
      .from("app_config").select("value").eq("key", ACADEMY.FLAG_KEY).maybeSingle();
    if (error) return false;
    return data?.value === true;
  } catch {
    return false;
  }
}
```

```js
import { makeRng, gaussian, STAT_KEYS } from "./fictionalRiderGenerator.js";
import { ACADEMY } from "./academyFlag.js";

function clamp(n, lo, hi) { return Math.max(lo, Math.min(hi, n)); }

// Ét intake-kuld: 3-5 kandidater, 1-3 markeret is_serious (høj potentiale-loft).
// identityBasis (team.season_1_identity_basis) biaser nation let. existingNames =
// foldede navne i DB (navne-unikhed). Returnerer [{ rider, is_serious }].
export function generateAcademyCandidates({ rng, referenceYear, existingNames, identityBasis = null }) {
  const count = ACADEMY.INTAKE_MIN + Math.floor(rng() * (ACADEMY.INTAKE_MAX - ACADEMY.INTAKE_MIN + 1));
  const seriousTarget = ACADEMY.SERIOUS_MIN + Math.floor(rng() * (ACADEMY.SERIOUS_MAX - ACADEMY.SERIOUS_MIN + 1));
  const out = [];
  for (let i = 0; i < count; i++) {
    const isSerious = i < seriousTarget;
    const age = clamp(Math.round(gaussian(rng, 18, 1.6)), ACADEMY.MIN_AGE, ACADEMY.MAX_AGE);
    const birthYear = referenceYear - age;
    // raw stats: lavere mean end seniorer (de skal UDVIKLE sig via akademiet)
    const statMean = isSerious ? 58 : 52;
    const stats = {};
    for (const k of STAT_KEYS) stats[k] = clamp(Math.round(gaussian(rng, statMean, 6)), 40, 85);
    // potentiale: seriøse højt loft (4.5-6.0), øvrige bredt (2.0-4.5)
    const pot = isSerious ? 4.5 + rng() * 1.5 : 2.0 + rng() * 2.5;
    const nationality_code = pickNation(rng, identityBasis); // genbrug generatorens vægt-fn + bias
    const { firstname, lastname } = pickUniqueName(rng, nationality_code, existingNames); // genbrug + foldNameNordic
    out.push({
      is_serious: isSerious,
      rider: {
        firstname, lastname, birthdate: `${birthYear}-06-15`, nationality_code,
        pcm_id: null, is_academy: false, team_id: null,
        potentiale: Math.round(pot * 2) / 2, // 0.5-trin
        ...stats,
      },
    });
  }
  return out;
}
```

```js
  const { count, error } = await supabase

// Orchestrator-trin: generér intake-kuld pr. MENNESKE-hold ved relaunch.
// Kører efter starter-squad (holdene findes), før kontrakt-seed. Idempotent:
// hold med eksisterende offered-kuld for den aktive sæson skippes.
export async function runAcademyIntake(supabase, { dryRun = true, seed = 2026, getManagerTeams } = {}) {
  if (!supabase?.from) throw new Error("Supabase client required");

  const teams = getManagerTeams
    ? await getManagerTeams(supabase)
    : (await import("./betaResetService.js")).getBetaManagerTeams
      ? await (await import("./betaResetService.js")).getBetaManagerTeams(supabase)
      : [];

  // aktiv sæson
  const seasonRes = await supabase.from("seasons").select("id, number, start_date").eq("status", "active").maybeSingle();
  if (seasonRes?.error) throw new Error(`runAcademyIntake sæson: ${seasonRes.error.message}`);
  const season = seasonRes?.data;
  if (!season) throw new Error("runAcademyIntake: ingen aktiv sæson");
  const referenceYear = Number(String(season.start_date || `${ACADEMY_REF_FALLBACK}`).slice(0, 4)) || ACADEMY_REF_FALLBACK;

  // navne-unikhed mod DB
  const existing = await fetchAllRows(() => supabase.from("riders").select("firstname, lastname").order("id"));
  const folded = new Set(existing.map((r) => foldNameNordic(`${r.firstname} ${r.lastname}`)));

  // hold der allerede har et kuld (idempotens)
  const seeded = await fetchAllRows(() => supabase.from("academy_intake").select("team_id").eq("season_id", season.id));
  const alreadySeeded = new Set(seeded.map((r) => r.team_id));

  let teamsSeeded = 0, candidatesCreated = 0;
  const rng = makeRng(seed);

  for (const team of teams) {
    if (alreadySeeded.has(team.id)) continue;
    const cands = generateAcademyCandidates({
      rng, referenceYear, existingNames: folded, identityBasis: team.season_1_identity_basis || null,
    });
    if (dryRun) { teamsSeeded++; candidatesCreated += cands.length; continue; }

    // insert riders → få id'er → insert academy_intake
    const riderPayload = cands.map((c) => c.rider);
    const { data: inserted, error } = await supabase.from("riders").insert(riderPayload).select("id");
    if (error) throw new Error(`academy riders insert (team ${team.id}): ${error.message}`);
    const intakeRows = inserted.map((r, i) => ({
      team_id: team.id, rider_id: r.id, season_id: season.id,
      is_serious: cands[i].is_serious, status: "offered",
    }));
    const { error: e2 } = await supabase.from("academy_intake").insert(intakeRows);
    if (e2) throw new Error(`academy_intake insert (team ${team.id}): ${e2.message}`);
    teamsSeeded++; candidatesCreated += cands.length;
  }

  return { dryRun, teams: teamsSeeded, candidates: candidatesCreated };
}
```
