Body font: DM Sans vs Inter Tight

Headings and the wordmark stay Bebas Neue either way. The only thing changing below is the body / UI font the whole app is set in. Same content, same layout, same gold. DM Sans is what runs in production today; Inter Tight is the proposed sibling that pairs with the Bebas wordmark and the Inter Tight Black CZ favicon. Read the paragraphs, then scan the numbers in the table (tabular figures) since this is a data-dense manager game.

DM Sans · humanist, friendly, slightly rounded. Geometric-humanist. Google default-ish.
Inter Tight · tighter Inter, condensed workhorse, neutral, dense-data friendly.

How to read this

DM Sans is warmer and a touch more rounded; it feels friendly and approachable. Slightly wider, so dense tables breathe a little more but fit a little less.

Inter Tight is more neutral and condensed; it packs data tighter and reads as more "tool / precise / pro analyst". It is the true sibling of the Bebas wordmark and the Inter Tight CZ favicon, so the whole system rhymes.

Trade-off: Inter Tight buys system-coherence and data-density (good for a manager game full of tables); DM Sans buys warmth and is a zero-cost no-op (already shipped). Either works. The brand-purist answer is Inter Tight; the pragmatic answer is keep DM Sans. Look at the numbers row in both and trust your gut.
